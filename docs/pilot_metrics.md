# Pilot Metrics

## Trust and quality

- expert agreement on `core_eval`
- false-positive rate on supportability
- percentage of answers with explicit evidence, inference, local validation status, and unresolved unknowns

## Speed

- median time to first evidence-backed answer

## Adoption

- number of real briefs assessed during pilot usage
- number of briefs that still require manual document hunting

## Phase-gate readiness

- whether hero briefs are stable enough to support a future Phase 2 blueprinting discussion
